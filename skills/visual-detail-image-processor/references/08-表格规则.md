# 表格规则

## 适用模块

- `03选尺码.jpg`
- `05尺码快选.jpg`

## 总原则

表格必须作为干净图层整块重建，再贴回 2000px 模板。不要在旧 JPG 表格上局部擦除、局部补白或局部补绿。

这样做是为了避免：

- 白色框块
- 绿色线条残留
- 旧文字残影
- 绿色表头出现白色竖线
- 圆角消失

## 03 选尺码

固定不动：

- `S3`
- `选尺码`
- 绿色底
- `Size`
- `/CM` 或 `/cm`
- 外部模板背景与圆角区域

动态替换：

- 表格列
- 表头
- 数据单元格

列规则：

- 先按素材表格识别出全部列。
- 不按字段名硬编码删除，例如不能只判断“肩宽”。
- 若某列全部数据为空，删除整列和表头。
- 若某个单格为空，只让该单格空白。
- 不根据其他列推算。

文字规则：

- 表头黑字。
- 数据黑字。
- 水平居中、垂直居中。
- 表头文字必须位于格子中间，不能偏上或偏下。
- 数字、英文、中文按字体规则混排并基线对齐。

圆角规则：

- 表格容器单独生成带透明通道的图层。
- 在图层上绘制圆角蒙版。
- 将表格图层贴回模板时保留透明通道。
- 不允许因为覆盖矩形导致圆角变直角。

## 05 尺码快选

固定不动：

- 标题 `尺码快选`
- `身高cm/体重kg`
- 外部模板背景与装饰

动态替换：

- 快选表格内容。

表头规则：

- 绿色表头白字。
- 绿色表头内部不能有白色竖线。
- 表头区域不能残留旧绿线或白色补丁。

表体规则：

- 表格内部全部黑字。
- 数字码值后面要加 `码`，例如 `110码`。
- 空值留空。
- 不使用红字。
- 表格右侧和底部保留安全边距，不画满到模板边缘。

## 空值判定

以下值视为空：

```text
""
"/"
"--"
"无"
"暂无"
"无数据"
```

## 验收

- 没有白框。
- 没有绿线残留。
- 没有旧 JPG 残影。
- 绿色表头无白色竖线。
- 圆角存在。
- 全空列已删除。
- 单格空值留空。
- 文字居中且字号统一。
