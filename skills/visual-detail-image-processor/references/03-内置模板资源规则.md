# 内置模板资源规则

## 默认模板目录

本 skill 自带 2000px 模板图片，默认目录为：

```text
template/
```

脚本默认读取这个目录，不需要每次传 `--template-dir`。

## 模板文件

目录内应包含：

```text
01产品信息.jpg
02透明图.jpg
03选尺码.jpg
04推荐尺码.jpg
05尺码快选.jpg
模特图1.jpg
模特图2.jpg
模特图2覆盖.png
模特图3.jpg
```

当前渲染脚本已使用：

```text
01产品信息.jpg
02透明图.jpg
03选尺码.jpg
05尺码快选.jpg
```

模特图模板和 `模特图2覆盖.png` 已作为模板资源保留，后续实现模特图渲染时必须使用同一目录。

## 外部模板覆盖

如果用户要换一套模板，可传：

```powershell
uv run python scripts/main.py `
  --input-root "<input-root>" `
  --output-root "<output-root>" `
  --template-dir "<custom-template-dir>"
```

传入 `--template-dir` 后，以外部模板目录为准。

## 约束

- 内置模板是 skill 的资源，不写死任何本机路径。
- 模板文件名必须保持一致，否则脚本应明确报错。
- 模板应保持 2000px 母版规格。
- 最终输出仍统一缩放到 790px 宽。
