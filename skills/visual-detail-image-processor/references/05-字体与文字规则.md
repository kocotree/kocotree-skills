# 字体与文字规则

## 字体

动态文字固定使用 skill 内置 `assets/` 下的静态字体文件，不再对外暴露字体参数：

- 中文字体：`assets/方正兰亭中黑_GBK-Regular.ttf`
- 字符 `1` 和 `%`：`assets/阿里妈妈方圆体-SemiBold.ttf`
- 其它数字和英文：`assets/G8321-MEDIUM.OTF`
- 斜杠 `/`：`assets/方正兰亭中黑_GBK-Regular.ttf`

阿里妈妈方圆体现在已提取为静态 `SemiBold` 文件，不再依赖运行时变体切换。

## 固定粗细

`assets/阿里妈妈方圆体-SemiBold.ttf` 是提取后的静态字体文件，直接使用即可；不会再在运行时切变体，也不会依赖系统字体状态。

## 提取静态字体文件

提取动作已经完成，后续只保留静态 TTF 文件，不再要求每次运行都提取。

## 混排

混排文本按字符选择字体，例如：

```text
110-170码
85%
身高cm
```

规则：

- `1` 使用阿里妈妈方圆体静态 SemiBold 文件。
- `%` 使用阿里妈妈方圆体静态 SemiBold 文件。
- 其它数字、英文、符号优先使用 `assets/G8321-MEDIUM.OTF`。
- 中文使用 `assets/方正兰亭中黑_GBK-Regular.ttf`。
- `/` 使用 `assets/方正兰亭中黑_GBK-Regular.ttf`。
- 全串按统一基线绘制，不能按各字体边界框顶部对齐。

## 字号

- 同一模块、同一层级文字使用固定字号。
- 不因为文本短就放大，不因为某个数字字体不同就改变字号。
- 只有超长文本确实超出安全宽度时，才允许整段等比例小幅缩小。
- 缩小只针对该段值，不影响其他同类字段。

## 空值

以下内容视为空值：

```text
""
"/"
"--"
"无"
"暂无"
"无数据"
```

空值处理：

- 产品信息：对应位置留空，不移动其他行。
- 表格单格：单格留空。
- 表格整列：若整列所有数据为空，删除该列和表头。
- 不推算、不补默认值。

## 对齐

- 产品信息右侧值按模板原示例文字中心位置放置。
- 产品信息右侧 `颜色` 值按单个颜色用 `/` 分割显示，斜杠使用中文字体并保持统一基线。
- 表格文字水平垂直居中。
- 透明图颜色名在同一水平基线，并居中于对应商品。
- 文本不能侵入固定模板文字或绿色底区域。
