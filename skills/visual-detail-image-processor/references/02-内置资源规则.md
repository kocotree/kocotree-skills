# 内置资源规则

## 字体资源

字体固定存放在 skill 根目录的 `assets/` 下：

```text
assets/
  方正兰亭中黑_GBK-Regular.ttf
  阿里妈妈方圆体-SemiBold.ttf
  G8321-MEDIUM.OTF
```

脚本固定读取这些字体，不再提供字体路径参数。

## 字体用途

- 中文：`assets/方正兰亭中黑_GBK-Regular.ttf`
- 字符 `1` 和 `%`：`assets/阿里妈妈方圆体-SemiBold.ttf`
- 其它数字、英文和常用符号：`assets/G8321-MEDIUM.OTF`

## 约束

- 字体文件缺失时脚本必须报错，不能退回默认字体。
- 不允许在命令行中传外部字体覆盖。
- 若要换字体，直接替换 `assets/` 中同名文件，并重新验证出图。
- 混排仍按统一 baseline 绘制，避免数字和中文上下跳动。
