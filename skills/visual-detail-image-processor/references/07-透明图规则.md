# 02 透明图规则

## 固定元素

`02透明图.jpg` 中以下内容固定不动：

- 绿色底
- `S2`
- `选颜色`
- 页面背景与固定装饰

只替换商品透明图和颜色名。

## 商品大小

同一张 `02透明图.jpg` 内，多张商品透明图必须视觉大小一致。

稳定做法：

1. 读取该商品 `透明图/` 下所有颜色 PNG。
2. 计算所有图的透明通道可见边界框。
3. 使用统一缩放基准，而不是每张图单独缩放到格子最大。
4. 让所有颜色图使用同一目标高度或同一最大可见高度约束。
5. 商品不能被拉伸，只能等比例缩放。

## 位置

- 商品放入等宽槽位。
- 每个商品以自身可见边界框在槽位中居中。
- 同页所有商品底部视觉高度尽量一致。
- 若款式不同导致高度差，优先保证整体视觉统一，不让某一张明显大或小。

## 颜色名

- 颜色名使用透明图文件名。
- 颜色名水平居中于对应商品槽位。
- 所有颜色名使用同一水平基线。
- 颜色名应贴近商品底部，不能离产品太远。

推荐计算：

```text
max_bottom = 所有贴入商品的可见底部最大值
label_y = max_bottom + 固定视觉间距
if label_y 超出安全区:
    整组商品上移
    重新计算 label_y
```

## 安全区

- 不能覆盖绿色底、`S2`、`选颜色`。
- 不能让颜色名贴到页面底部。
- 若商品较高，先整体上移，不单独压缩某个颜色。

## 常见问题

- 文字太远：通常是颜色名使用了固定 y，而没有根据商品真实底部计算。
- 商品大小不一致：通常是每张透明图单独按可见边界框填满槽位。
- 某个颜色明显偏移：通常是没有用透明通道可见边界框做视觉居中。
