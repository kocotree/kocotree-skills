# 模特图规则

## 适用模块

- `模特图1.jpg`
- `模特图2.jpg`
- `模特图3.jpg`

## 固定元素

只替换照片框内部。以下内容固定保留：

- 模板背景
- 标题
- 标志
- 装饰线
- 绿色装饰
- 遮罩和覆盖层
- 所有固定文字

图片模型只能处理照片框内部，不能处理模板文字、标志、表格、绿色装饰或覆盖层。

## 构图优先级

模特图裁切和扩图的优先级：

```text
头部完整 > 主体身体完整 > 服装展示完整 > 脚部完整
```

脚在必要时可以不完整露出；头部和主体不能被切坏。

## 基础构图

- 不拉伸、不变形，只能等比例缩放。
- 尽量展示完整模特和完整服装。
- 头顶到画框顶部保留合理空间。
- 左右留安全边距，人物不能贴边。
- 主体尽量居中，但允许根据动作略微偏移。
- 如果人物动作横向展开，优先保手臂和衣服轮廓，脚可以被裁。
- 如果原图是半身或脚本来缺失，不强行补造脚。

## 代码裁切策略

优先使用代码完成稳定合成：

1. 读取原模特图。
2. 使用当前脚本中手动校准的照片框坐标作为最终替换区域。
3. 如果存在 `visual_model_selection` 坐标，优先按该坐标裁切源图。
4. 如果没有坐标、坐标无效或源文件不存在，再根据目标照片框比例执行默认 `cover_crop`。
5. 裁切后等比例缩放，最终必须填满整个坐标框，不允许留白或等比完整包含缩放。
6. 使用圆角蒙版贴回模板。

## 智能审美选框

`visual_model_selection` 是模特图的智能审美裁切数据，不使用图像生成能力。智能模型只输出源图原始像素坐标，脚本负责裁切、缩放和模板合成。

当商品目录存在模特图素材时，必须把智能审美选图/选框作为渲染前置步骤：先为每张可用模特图选择更适合模板照片框的源图区域，输出 `crop_box` 到 `visual_model_selection` JSON，再运行渲染脚本。脚本不会停下来等待选框数据；如果渲染前没有坐标 JSON，它会直接回退默认 `cover_crop`。

只有在无法读取源图、无法可靠判断主体、用户明确要求跳过，或当前环境不支持视觉判断时，才允许不生成坐标并回退脚本默认 `cover_crop`，并在报告或回复中说明原因。

默认目录：

```text
scripts/output/visual_model_selection/
```

脚本会自动读取最新的 `visual_model_selection-*.json`，也可通过 `--visual-model-selection` 指定文件。坐标结构和规则见 `references/10-智能模特图选框规则.md`。

批处理时推荐始终显式传入本轮生成的坐标文件：

```powershell
uv run python main.py `
  --input-root "<input-root>" `
  --output-root "<output-root>" `
  --visual-model-selection "output/visual_model_selection/visual_model_selection-<时间戳>.json"
```

## 当前照片框坐标

坐标以 2000px 模板母版为准，格式为：

```text
(left, top, right, bottom)
```

当前脚本使用的实际坐标：

```text
模特图1：24, 550, 1974, 3296
模特图2：260, 216, 1700, 2224
模特图3：24, 98, 1974, 2844
```

圆角半径统一为 `52`。替换时照片会填满这些坐标框；如果源图比例与坐标框不同，只能裁切多余部分，不能拉伸、不能露出模板底图。

验证坐标时，在 2000px 模板上检查这些矩形是否刚好贴合照片框内部边界；最终输出到 790px 前不再重新计算坐标。

## 主体检测与构图评分

可用启发式评分判断裁切是否合格：

- 头顶距顶部是否在安全范围。
- 主体边界框左右是否有边距。
- 人物主体是否超过框宽过多。
- 裁切后是否丢头、丢上身、丢服装主体。
- 脚部丢失只作为低优先级扣分。

若无法可靠检测主体，应使用保守中心裁切，并人工报告风险。

## 图片模型辅助

图片模型只在以下情况使用：

- 原图比例与照片框差距太大，需要扩图补边。
- 代码裁切会切头、切身体或服装主体。
- 背景需要在照片框内部自然延展。

限制：

- 模型输出只作为照片框内部图。
- 最终模板合成仍由代码完成。
- 不允许让图片模型重画整张详情页。
- 模型结果如果改坏人物、服装、动作或产生异常背景，回退代码裁切版本。

## 模特图2覆盖

`模特图2` 有特殊规则：

1. 先在 `模特图2.jpg` 的照片框内部贴入纯照片底图。
2. 最后把 `<template-dir>/模特图2覆盖.png` 按 2000px 一比一贴到最上层。
3. 覆盖层不缩放、不移动、不重绘。
4. 覆盖后再把整图等比例缩到 790px。

这样做是为了避免绿色线条和装饰层错乱。

## 参考图

用户可能提供模特动作参考图目录。参考图用于理解商业童装模特动作审美，不是固定模板：

- 参考头部留白比例。
- 参考主体在竖框中的完整度。
- 参考服装展示优先级。
- 不要求每款角度完全一致。

## 验收

- 人物不变形。
- 头部完整。
- 主体衣服展示清楚。
- 顶部留白自然。
- 左右不贴边。
- 圆角存在。
- 固定模板装饰未被破坏。
- `模特图2覆盖.png` 在最上层完整覆盖。
