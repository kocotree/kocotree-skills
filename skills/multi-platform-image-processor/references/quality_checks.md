# 自动质检规则

## 基础检查

每张输出图片都检查：

- 文件是否存在。
- 格式是否符合目录规则。
- 宽高是否符合目录规则。
- 文件大小是否超过限制。
- PNG 透明图是否保留 alpha 通道。
- PNG 透明图是否经过 `pngquant` 压缩并小于平台大小限制。
- 详情页命名是否连续。

## 平台重点

### 天猫通用版

- `790详情页` 宽度必须是 `790px`，高度应小于等于 `1600px`。
- `800透明图` 必须为 PNG 且保留透明通道。
- `sku\1440` 没有来源时允许为空。

### CBME

- `750主图` 必须是 `750x750` JPG。
- `750详情页` 宽度必须是 `750px`。

### 蜂享家＋爱库存

- `790详情页` 宽度必须是 `790px`。
- 单张高度应小于等于 `4800px`。
- 数量应小于等于 `20`。
- 单张大小应小于等于 `1MB`。

### 京东

- `透明图` 必须是 `800x800 PNG`。
- `透明图` 必须保留 alpha 通道。
- `详情页` 命名必须从 `601.jpg` 连续。

### 唯品会

- `1200主图` 必须是 `1200x1200 JPG`。
- `1200透明图` 必须是 PNG，至少一边为 `1200px`。
- `750详情页` 宽度必须是 `750px`。

### 站外通用版

- `800sku去除文字` 必须输出 JPG。
- `800白底图＋logo` 必须输出 JPG。
- `790详情页去除文字` 和 `800主图去除文字和边框` 允许为空。

## 报告级别

- `警告`：尺寸、大小、数量等可量化规则不完全满足，但文件已输出。
- `风险`：模型去字降级、低置信度或视觉效果可能不完美。
- `失败项`：文件无法读取、无法写出、关键步骤无法完成。
